本地服务器

dev server

boi提供一个本地的服务器，并且集成了Mock功能，以便FE可以不依赖后端接口实现独立开发。

在项目根目录下执行boi serve开启本地服务器，开启成功后会有以下提示：

==> Server is listening localhost:8888

本地服务器默认监听8888端口，可以通过boi.serveAPI修改配置，如下：

boi.serve({
  domain: 'localhost',
  port: '8000'
});

domain推荐不要指定，默认为空。如果指定为localhost的话便不能通过IP访问。
