Image编译配置

Image编译配置

Image编译配置使用boi.specAPI：

boi.spec('image',<options>);

第一个参数代表需要配置的模块，Image对应value为image。

options包括以下可配置项：

  • extType：Array，图片后缀名数组，默认值为['jpg','png']；

  • destDir：String，编译输出的目录，相对于[basic](config.basic.md).localPath.dest。默认为'image'；

  • base64：Boolean，是否对小体积图片使用base64嵌入，默认为false；

  • base64Limit：Number，参与base64编码的文件体积上限，以字节为单位，默认为10000。此配置项单独配置无效，必须在base64配置项为true时才会生效。

  • useHash：Boolean，编译输出的文件名是否加上hash指纹，默认为true。
